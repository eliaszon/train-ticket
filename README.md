## 🚝 This is a train ticket web application based on React.

### Main Used：

- React(React Hooks)
- Redux
- React-Redux
- dayjs
- urijs

### Start

```
git clone git@github.com:Mexion/train-ticket.git
```

CD to the folder,use start to open DevServer:

```
npm run start
```

or use build to build the package:

```
npm run build
```
